Real-time call transcription and summary tool

Problem

Transcription is not readily available for many calls credit analysts rely on. Earnings calls for unlisted companies, NetRoadshow presentations, audio calls with company management, and Zoom or Teams meetings with external parties often have no transcript. Capturing them means a human note-taker and hours of manual write-up.

What I built

A Python program that records live audio input and output and transcribes it in real time using AssemblyAI speech-to-text. The transcript is sent automatically to Claude, which adds speaker labels and produces a clean call summary. Output is a labelled transcript plus a ready-to-use meeting summary, with no manual note-taking.

Stack
Python AssemblyAI Claude API
Result

Removed the need for a human note-taker on management calls and non-recorded presentations. Frees the analyst to focus on the call itself, and delivers a summary the moment the call ends.
